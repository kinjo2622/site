# 沖縄そば専門店 すばやー 公式サイト

沖縄そば専門店「すばやー」の公式ウェブサイトのソースコードです。
日本語・英語の2言語に対応した静的サイトで、メニュー紹介・店舗アクセス・EC サイトへの導線を提供します。

## 使用技術

- HTML5
- CSS3
- JavaScript
- jQuery
- Slick Carousel
- Google Fonts

## ディレクトリ構成

```
site-main/
├── index.html          # トップページ（日本語）
├── menu.html           # メニューページ（日本語）
├── access.html         # アクセスページ（日本語）
├── ec-site.html        # EC サイトページ
├── en-index.html       # トップページ（英語）
├── en-menu.html        # メニューページ（英語）
├── en-access.html      # アクセスページ（英語）
├── CSS/
│   ├── common.css      # 共通スタイル
│   ├── top-style.css   # トップページ用スタイル
│   ├── menu.css        # メニューページ用スタイル
│   └── access.css      # アクセスページ用スタイル
├── JS/
│   ├── top-script.js       # トップページ用スクリプト（日本語）
│   └── en-top-script.js    # トップページ用スクリプト（英語）
└── images/             # 画像素材
```

## ページ一覧

| ファイル | 言語 | 内容 |
|---|---|---|
| `index.html` | 日本語 | トップページ（スライダー・こだわり紹介・バナー等） |
| `menu.html` | 日本語 | メニュー一覧（沖縄そば各種・サイドメニュー・デザート） |
| `access.html` | 日本語 | 店舗所在地・アクセス情報 |
| `ec-site.html` | 日本語 | EC サイト（オンライン販売）への案内 |
| `en-index.html` | English | Top page |
| `en-menu.html` | English | Menu |
| `en-access.html` | English | Access / Location |
